## 💡 使用方式

### 🔍 项目复刻模式
> *"我想复刻 XXX 项目"*

系统自动匹配最相关的UP主和视频，提供详细步骤指南和资源链接。

### 💬 技术咨询模式
> *"如何实现 XXX 功能"*

从知识库提取最佳实践，结合多位UP主经验给出建议。

### 🗺️ 学习规划模式
> *"我想学习 XXX 技术"*

生成个性化学习路线图，推荐对应视频和项目。

### 🔧 问题诊断模式
> *遇到技术问题时*

从知识库查找类似案例，提供解决方案和调试建议。
