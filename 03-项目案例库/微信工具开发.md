## 二、FREEWECHATPUSH（⭐97）

### 项目介绍
技术爬爬虾开源的免费微信消息推送工具，基于企业微信API实现。

### 功能
- 通过企业微信发送消息到微信
- 支持文本、Markdown、图片、文件消息
- 支持@指定人
- 简单的API调用接口

### 使用步骤

#### Step 1：创建企业微信应用

1. 注册企业微信（https://work.weixin.qq.com）
2. 创建自建应用
3. 获取：Corpid、Corpsecret、Agentid
4. 配置可信IP

#### Step 2：配置API

```python
import requests
import json

class WeChatPush:
    def __init__(self, corpid, corpsecret, agentid):
        self.corpid = corpid
        self.corpsecret = corpsecret
        self.agentid = agentid
        self.access_token = None
    
    def get_access_token(self):
        url = f"https://qyapi.weixin.qq.com/cgi-bin/gettoken"
        params = {
            "corpid": self.corpid,
            "corpsecret": self.corpsecret
        }
        response = requests.get(url, params=params)
        data = response.json()
        self.access_token = data["access_token"]
        return self.access_token
    
    def send_text(self, content, touser="@all"):
        url = f"https://qyapi.weixin.qq.com/cgi-bin/message/send"
        params = {"access_token": self.access_token}
        
        payload = {
            "touser": touser,
            "msgtype": "text",
            "agentid": self.agentid,
            "text": {
                "content": content
            }
        }
        
        response = requests.post(url, params=params, json=payload)
        return response.json()
    
    def send_markdown(self, content, touser="@all"):
        url = f"https://qyapi.weixin.qq.com/cgi-bin/message/send"
        params = {"access_token": self.access_token}
        
        payload = {
            "touser": touser,
            "msgtype": "markdown",
            "agentid": self.agentid,
            "markdown": {
                "content": content
            }
        }
        
        response = requests.post(url, params=params, json=payload)
        return response.json()

# 使用示例
push = WeChatPush(
    corpid="your_corpid",
    corpsecret="your_corpsecret",
    agentid=your_agentid
)
push.get_access_token()
push.send_text("巡检完成！发现3条新通知。")
```

## 四、实战：巡检系统微信推送集成

### 架构设计

```
巡检脚本运行
    ↓
检测到新通知
    ↓
生成巡检摘要
    ↓
调用企业微信API
    ↓
推送到微信
```

### 完整实现

```python
import requests
import json
from datetime import datetime

class InspectorNotifier:
    """巡检结果微信通知器"""
    
    def __init__(self, webhook_url):
        self.webhook_url = webhook_url
    
    def notify(self, new_items, total_items):
        """发送巡检结果通知"""
        if not new_items:
            return  # 没有新通知则不推送
        
        # 构建Markdown消息
        content = self._build_message(new_items, total_items)
        
        # 发送消息
        payload = {
            "msgtype": "markdown",
            "markdown": {"content": content}
        }
        
        response = requests.post(self.webhook_url, json=payload)
        return response.json()
    
    def _build_message(self, new_items, total_items):
        """构建通知消息"""
        now = datetime.now().strftime("%Y-%m-%d %H:%M")
        
        lines = [
            f"## 🔍 合工大官网巡检报告",
            f"> 时间：{now}",
            f"> 新通知：**{len(new_items)}** 条",
            f"> 总索引：**{total_items}** 条",
            "",
            "### 📋 新通知列表",
        ]
        
        for i, item in enumerate(new_items[:10], 1):  # 最多展示10条
            lines.append(f"{i}. [{item['title']}]({item['url']})")
            lines.append(f"   栏目：{item['category']} | 时间：{item['date']}")
        
        if len(new_items) > 10:
            lines.append(f"\n> 还有 {len(new_items)-10} 条通知未展示...")
        
        lines.append("")
        lines.append("[查看完整报告](http://your-server/report)")
        
        return "\n".join(lines)

# 使用示例
notifier = InspectorNotifier(webhook_url="YOUR_WEBHOOK_URL")

# 模拟巡检结果
new_items = [
    {"title": "关于2026年暑期实习的通知", "url": "...", "category": "教务处", "date": "2026-04-28"},
    {"title": "计算机学院奖学金评定通知", "url": "...", "category": "学院通知", "date": "2026-04-27"},
]

notifier.notify(new_items=new_items, total_items=151)
```

## 五、进阶：定时自动巡检+推送

### 方案A：GitHub Actions定时运行

```yaml
# .github/workflows/inspector.yml
name: Daily Inspector

on:
  schedule:
    - cron: '0 0 * * *'  # 每天UTC 00:00（北京时间08:00）

jobs:
  inspect:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      
      - name: Setup Python
        uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      
      - name: Install dependencies
        run: pip install -r requirements.txt
      
      - name: Run inspector
        env:
          WECHAT_WEBHOOK_URL: ${{ secrets.WECHAT_WEBHOOK_URL }}
        run: python inspector.py
      
      - name: Deploy report
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./output
```

### 方案C：本地定时任务

```python
# scheduler.py
import schedule
import time
from inspector import run_inspection
from notifier import InspectorNotifier

def daily_task():
    result = run_inspection()
    notifier = InspectorNotifier(webhook_url="YOUR_WEBHOOK_URL")
    notifier.notify(
        new_items=result['new_items'],
        total_items=result['total_items']
    )

# 每天早上8点执行
schedule.every().day.at("08:00").do(daily_task)

while True:
    schedule.run_pending()
    time.sleep(60)
```

## 六、安全注意事项

### API安全
- ⚠️ Webhook URL不要泄露（相当于密码）
- ⚠️ 不要将API Key提交到GitHub
- ⚠️ 使用GitHub Secrets存储敏感信息

### 使用规范
- 不要频繁推送（避免被标记为骚扰）
- 推送内容要有价值（避免信息噪音）
- 提供取消订阅的选项
