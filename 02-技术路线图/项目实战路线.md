## 阶段一：新手项目（1-3周）

### 项目1：Windows系统优化与效率工具配置
- **难度**：⭐
- **来源UP主**：小in分享
- **目标**：打造高效的个人电脑工作环境
- **所需工具**：SpaceSniffer, Everything, Snipaste, VS Code, Obsidian
- **步骤**：
  1. 执行C盘清理（参考小in分享教程）
  2. 安装5+效率工具并配置
  3. 制作随身工具U盘（PortableApps）
  4. 配置VS Code常用插件
- **产出**：一台优化过的、效率工具齐全的开发电脑
- **可复刻视频**：小in分享「C盘清理8个位置」「便携软件U盘方案」

### 项目2：GitHub个人主页搭建
- **难度**：⭐⭐
- **来源UP主**：技术爬爬虾
- **目标**：建立个人代码托管空间
- **所需工具**：Git, GitHub, VS Code
- **步骤**：
  1. 注册GitHub账号
  2. 学习Git基本操作（跟着《小白玩转Git/GitHub》）
  3. 创建个人仓库，编写README
  4. 配置SSH Key
  5. 第一次push代码
- **产出**：GitHub个人主页 + 第一个代码仓库
- **可复刻视频**：技术爬爬虾《小白玩转Git/GitHub》

### 项目3：AI对话助手使用
- **难度**：⭐
- **来源UP主**：GenJi
- **目标**：学会高效使用AI对话工具
- **所需工具**：ChatGPT, Claude, Kimi
- **步骤**：
  1. 注册并体验3个AI对话平台
  2. 学习GenJi的prompt方法论
  3. 用AI完成3个实际任务（写作/翻译/编程）
  4. 建立个人prompt库
- **产出**：个人prompt库 + AI使用心得
- **可复刻视频**：GenJi「ChatGPT深度解析」「AI大模型原理」

## 阶段二：进阶项目（3-6周）

### 项目4：Docker部署Gemini对话网站
- **难度**：⭐⭐⭐
- **来源UP主**：技术爬爬虾
- **目标**：在国内部署一个可用的Gemini对话网站
- **所需工具**：Docker, Next.js, Vercel, Google AI API
- **步骤**：
  1. 国内安装Docker（使用DOCKER_INSTALLER）
  2. Fork GEMINI-PLAYGROUND仓库
  3. 配置Google AI API Key
  4. Vercel一键部署
  5. 配置自定义域名（可选）
- **产出**：个人Gemini对话网站
- **可复刻资源**：https://github.com/tech-shrimp/gemini-playground

### 项目5：n8n自动化工作流搭建
- **难度**：⭐⭐⭐
- **来源UP主**：小in分享
- **目标**：构建个人自动化工作流
- **所需工具**：n8n, Docker, Ollama
- **步骤**：
  1. Docker部署n8n
  2. 本地部署Ollama（可选AI模型）
  3. 构建RSS新闻聚合工作流
  4. 构建AI聊天机器人工作流
  5. 构建定时任务工作流
- **产出**：3个以上n8n自动化工作流
- **可复刻视频**：小in分享「n8n本地AI部署教程」

### 项目6：AI绘画作品集
- **难度**：⭐⭐⭐
- **来源UP主**：GenJi
- **目标**：掌握AI绘画工具，产出高质量作品
- **所需工具**：Midjourney 或 Stable Diffusion, ComfyUI
- **步骤**：
  1. 选择AI绘画工具（MJ或SD）
  2. 学习基础prompt编写
  3. 跟着GenJi教程练习
  4. 用ComfyUI搭建工作流
  5. 创作10+张高质量作品
- **产出**：个人AI绘画作品集
- **可复刻视频**：GenJi「AI动画制作课程」「SD花活集」
