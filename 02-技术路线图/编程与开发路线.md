## 阶段一：零基础 → 入门（1-2周）

### 学习目标
- 掌握Git/GitHub基本操作
- 学会Python基础编程
- 建立编程开发的认知框架

### 推荐资源

| 资源 | 来源 | 难度 | 关键词 |
|------|------|------|--------|
| 《小白玩转Git/GitHub》课程 | 技术爬爬虾 | ⭐⭐ | Git、GitHub、版本控制 |
| GitHub Actions入门 | 技术爬爬虾 | ⭐⭐ | CI/CD、自动化、GitHub |
| Python基础教程 | 在线资源 | ⭐⭐ | Python、编程入门 |

### 必备工具

| 工具 | 用途 | 获取方式 |
|------|------|----------|
| Git | 版本控制 | https://git-scm.com |
| GitHub | 代码托管 | https://github.com |
| VS Code | 代码编辑器 | https://code.visualstudio.com |
| Python | 编程语言 | https://python.org |

### 实践项目
- [ ] 注册GitHub账号，创建第一个仓库
- [ ] 完成一次commit、push、pull操作
- [ ] 写一个Python脚本（如文件重命名、数据统计）
- [ ] Fork技术爬爬虾的一个项目，本地运行

### 避坑指南
- ❌ 不要把密码和密钥提交到GitHub
- ❌ 不要直接在main分支上开发
- ✅ 学会使用.gitignore文件
- ✅ commit message写清楚改了什么

---

## 阶段二：入门 → 进阶（2-4周）

### 学习目标
- 掌握Docker容器化部署
- 学会API开发和代理
- 理解CI/CD自动化

### 推荐资源

| 资源 | 来源 | 难度 | 关键词 |
|------|------|------|--------|
| DOCKER_INSTALLER | 技术爬爬虾 | ⭐⭐ | Docker安装、国内镜像 |
| DOCKER_IMAGE_PUSHER | 技术爬爬虾 | ⭐⭐ | 镜像转存、GitHub Actions |
| GITHUBACTIONSAMPLE | 技术爬爬虾 | ⭐⭐ | GitHub Actions、自动化 |
| Cloudflare Worker入门 | 技术爬爬虾 | ⭐⭐⭐ | 边缘计算、API代理 |
| Claude Code入门 | 小天fotos | ⭐⭐⭐ | AI编程、Claude |

### 必备工具

| 工具 | 用途 | 获取方式 |
|------|------|----------|
| Docker Desktop | 容器化 | https://docker.com |
| Claude Code | AI编程 | Anthropic |
| Cloudflare账号 | 边缘计算 | https://cloudflare.com |
| Postman | API测试 | https://postman.com |

### 实践项目
- [ ] 国内安装Docker，运行第一个容器
- [ ] 配置Docker镜像转存到阿里云
- [ ] 创建一个GitHub Actions工作流（如定时运行脚本）
- [ ] 部署一个Cloudflare Worker（如简单的API代理）

### 避坑指南
- ❌ Docker Desktop占用资源较多，注意内存分配
- ❌ Cloudflare Worker有免费额度限制（10万次/天）
- ✅ 用docker-compose管理多容器应用
- ✅ GitHub Actions的secrets用来存储敏感信息
