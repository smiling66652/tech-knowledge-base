### 目标
掌握AI Agent的开发和部署，构建智能自动化系统。

### 学习路径

```
第1周：深入Claude Code
  → 学习Skill开发（小天fotos教程）
  → 开发第一个自定义Skill
  → 理解Agent架构设计

第2周：MCP Server开发
  → 学习MCP协议（小天fotos + 技术爬爬虾）
  → 用TypeScript开发第一个MCP Server
  → 在Claude Desktop中配置使用

第3-4周：Agent系统设计
  → 学习Agent Teams（小天fotos）
  → 设计多Agent协作系统
  → 实现一个完整的Agent项目
```

### 推荐资源
- 小天fotos：Claude Code Skill完整教程
- 小天fotos：MCP协议深度解析
- 小天fotos：Agent Teams蜂群模式
- 技术爬爬虾：15个好用MCP推荐
- 技术爬爬虾：AGENT-SKILLS-EXAMPLES

### 实战项目
- 开发一个文件管理Skill
- 开发一个网页内容抓取MCP
