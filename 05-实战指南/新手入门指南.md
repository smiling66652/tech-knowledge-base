# 新手入门指南

> 综合四位UP主内容，面向零基础用户
> 创建时间：2026-04-28

---

## 写在前面

如果你是第一次接触AI和编程技术，不要担心！这份指南会带你从零开始，循序渐进地建立技术能力。

**核心原则**：先学会用，再理解原理，最后深入开发。

---

## 第一步：优化你的电脑（1天）

### 为什么先做这个
一台流畅的电脑是所有技术学习的基础。

### 操作清单

1. **清理C盘空间**
   - 参考小in分享「C盘清理8个位置」
   - 预计释放5-20GB空间
   - 工具：SpaceSniffer + 手动清理

2. **安装基础效率工具**
   - Everything（文件搜索）
   - Snipaste（截图）
   - VS Code（代码编辑）
   - Windows Terminal（终端）

3. **制作随身工具U盘**
   - 参考小in分享「便携软件U盘方案」
   - 工具：PortableApps

### 验证标准
- [ ] C盘剩余空间 > 20GB
- [ ] 安装了5个以上效率工具
- [ ] VS Code可以正常打开和编辑文件

---

## 第二步：体验AI工具（2-3天）

### 为什么先做这个
AI工具是当前最重要的技术方向，先体验才能理解后续要学的内容。

### 操作清单

1. **注册AI对话工具**
   - ChatGPT（https://chat.openai.com）— 需要科学上网
   - Claude（https://claude.ai）— 需要科学上网
   - Kimi（https://kimi.moonshot.cn）— 国内免费
   - 通义千问（https://tongyi.aliyun.com）— 国内免费

2. **尝试以下任务**
   - 用AI帮你写一封邮件
   - 用AI翻译一段英文
   - 用AI解释一个你不理解的概念
   - 用AI帮你规划一周的学习计划

3. **学习Prompt技巧**
   - 参考GenJi「ChatGPT深度解析」
   - 学会写清晰的指令

### 验证标准
- [ ] 能流畅使用2个以上AI对话工具
- [ ] 写出了3个有效的prompt
- [ ] 体验过AI辅助写作/翻译/编程

---

## 第三步：建立代码能力（1-2周）

### 为什么做这个
编程是AI时代的核心技能，即使不成为程序员，也需要基本的编程能力。

### 操作清单

1. **注册GitHub账号**
   - 参考《小白玩转Git/GitHub》（技术爬爬虾）

2. **学习Git基础**
   - clone / add / commit / push
   - 创建仓库、分支管理

3. **Python入门**
   - 安装Python 3.11+
   - 学习基本语法：变量、条件、循环、函数
   - 写一个简单的脚本（如文件批量重命名）

4. **用Claude辅助编程**
   - 遇到问题先问AI
   - 理解AI给出的代码
   - 自己修改和调试

### 验证标准
- [ ] 拥有GitHub账号并有第一个仓库
- [ ] 能用Git进行基本的版本控制
- [ ] 写出3个以上Python脚本
- [ ] 能用AI辅助解决编程问题

---

## 第四步：探索AI编程（2-3周）

### 为什么做这个
AI编程是当前最热门的技术方向，可以大幅提升开发效率。

### 操作清单

1. **了解AI编程工具**
   - Claude Code（小天fotos推荐）
   - Cursor（AI代码编辑器）

2. **学习MCP协议概念**
   - 参考小天fotos「MCP协议深度解析」
   - 安装2-3个MCP Server

3. **体验AI辅助开发**
   - 用AI帮你在GitHub上Fork一个项目
   - 用AI帮你理解项目代码
   - 用AI帮你修改和优化代码

### 验证标准
- [ ] 了解MCP协议的基本概念
- [ ] Claude Desktop安装了MCP Server
- [ ] 用AI完成一个小型编程任务

---

## 第五步：构建第一个自动化项目（2-3周）

### 为什么做这个
将前面学到的知识整合成一个实际项目，是巩固学习效果的最佳方式。

### 操作清单

1. **安装Docker**
   - 参考技术爬爬虾 DOCKER_INSTALLER
   - 参考小in分享「WSL2/Docker Desktop入门」

2. **部署n8n**
   - 参考小in分享「n8n本地AI部署教程」
   - Docker一键部署n8n

3. **构建自动化工作流**
   - 在n8n中创建第一个工作流
   - 如：RSS新闻聚合 → AI摘要 → 微信推送

### 验证标准
- [ ] Docker正常运行
- [ ] n8n可以访问
- [ ] 构建了1个以上自动化工作流

## 常见问题

### Q: 我没有编程基础，能学吗？
A: 当然可以！GenJi和小in分享的内容都是面向零基础用户的。先从AI工具使用开始，再逐步学习编程。

### Q: 需要科学上网吗？
A: 部分工具需要（如ChatGPT、Claude、Midjourney）。国内也有替代品：Kimi、通义千问等。

### Q: 需要买什么设备？
A: 普通笔记本即可。如果要做AI绘画本地部署，需要至少8GB显存的GPU。

### Q: 每天需要投入多少时间？
A: 建议1-2小时。重要的是持续，而不是一次学很多。

### Q: 学完能做什么？
A: 可以用AI辅助工作、自动化日常任务、开发个人项目、提升就业竞争力。
